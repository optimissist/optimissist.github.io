NOTES:

  • Named colors — English words that describe colors, also called keyword colors
  • RGB — numeric values that describe a mix of red, green, and blue
  • HSL — numeric values that describe a mix of hue, saturation, and lightness
  • You can add opacity to a color by adding "a" to hsl or rgb (hsla or rgba) and adding a value between 0 and 1. 0 is completely transparent.
  • You can change the opacity of a hexadecimal color. Hex opacity ranges from 00 (transparent) to FF (opaque).
